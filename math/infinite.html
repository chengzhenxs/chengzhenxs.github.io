定义

无穷小

如果函数 f ( x ) 当 x → x 0 （或 x → ∞ ）时的极限为零，那么称函数 f ( x ) 为当 x → x 0 （或 x → ∞ ）时的无穷小。

无穷大

设函数 f ( x ) 在 x 0 的某一去心邻域内有定义（或 | x | 大于某一正数时有定义）。如果对于任意给定的正数 M ，总存在对应的正数 δ （或正数 X ），只要 x 适合不等式 0 < | x - x 0 | < δ （或 | x | > X ），对应的函数值 f ( x ) 总满足不等式

| f ( x ) | < M

则称函数 f ( x ) 为当 x → x 0 （或 x → ∞ ）时的无穷大。
