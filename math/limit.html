自变量趋于有限值时的极限

设函数 f ( x ) 在 x 0 的某一去心邻域1内有定义。如果存在常数 A ，对于任意给定的正数 ε ，总存在正数 δ ，使得当 x 满足不等式 0 < | x - x 0 | < δ 时，对应的函数值 f ( x ) 都满足不等式

| f ( x ) - A | < ε

那么常数 A 就叫做函数 f ( x ) 当 x → x 0 时的极限，记作

lim x → x 0 f ( x ) = A 或 f ( x ) → A （当 x → x 0 ）

自变量趋于无穷大时的极限

| f ( x ) - A | < ε

那么常数 A 就叫做函数 f ( x ) 当 x → ∞ 时的极限，记作

lim x → ∞ f ( x ) = A 或 f ( x ) → A （当 x → ∞ ）

单侧极限

左极限

设函数 f ( x ) 在 x 0 的某一去心邻域内有定义。如果存在常数 A ，对于任意给定的正数 ε ，总存在正数 δ ，使得当 x 满足不等式 x 0 - δ < x < x 0 时，对应的函数值 f ( x ) 都满足不等式

| f ( x ) - A | < ε

那么常数 A 就叫做函数 f ( x ) 当 x → x 0 时的左极限，记作

lim x → x 0 - f ( x ) = A 或 f ( x 0 - ) = A

右极限

设函数 f ( x ) 在 x 0 的某一去心邻域内有定义。如果存在常数 A ，对于任意给定的正数 ε ，总存在正数 δ ，使得当 x 满足不等式 x 0 < x < x 0 + δ 时，对应的函数值 f ( x ) 都满足不等式

| f ( x ) - A | < ε

那么常数 A 就叫做函数 f ( x ) 当 x → x 0 时的右极限，记作

lim x → x 0 + f ( x ) = A 或 f ( x 0 + ) = A

左极限与右极限统称为单侧极限。

函数 f ( x ) 当 x → x 0 时极限存在的充分必要条件是 f ( x 0 - ) = f ( x 0 + ) 。

性质

如果 lim x → x 0 f ( x ) 存在，那么这个极限唯一。

如果 lim x → x 0 f ( x ) = A 且 lim x → x 0 f ( x ) = B ，则 A = B 。

如果 lim x → x 0 f ( x ) = A ，那么存在常数 M > 0 和 δ > 0 ，使得当 0 < | x - x 0 | < δ

时，有 | f ( x ) | ≤ M 。

如果 lim x → x 0 f ( x ) = A ，且 A > 0 （或 A < 0 ），那么存在常数 δ > 0 ，使得当 0 < | x - x 0 | < δ 时，有 | f ( x ) | > 0 （或 | f ( x ) | < 0 ）。

如果在 x 0 的某去心邻域内 f ( x ) ≥ 0 （或 f ( x ) ≤ 0 ），而且 lim x → x 0 f ( x ) = A ，那么 A ≥ 0 （或 A ≤ 0 ）。

如果 lim x → x 0 f ( x ) = A （ A ≠ 0 ），那么就存在着 x 0 的某一去心邻域 U ∘ ( x x ) ，当 x ∈ U ∘ ( x x ) 时，就有 | f ( x ) | > | A | 2 。

1点 a 的去心 δ 邻域记作 U ∘ ( a , δ ) ，且 U ∘ ( a , δ ) = { x | 0 < | x - a | < δ } 。
